# Proyecto Gestión de Ninjas - Arquitectura Monolitica 
Grupo Tomates:
* Natalia Osejo
* Cristian Reyes

Aplicación monolítica desarrollada en Spring Boot que gestiona entidades de Ninjas, Aldeas, Jutsus y Misiones.

## Requesitos para correr el proyecto
* Java 17 o superior
* Navegador web para acceder al frontend (index.html)

## Pasos para ejecutar la aplicación
* Clonar el repositorio (git clone [https://github.com/crisr2/proyecto-monolito-naruto](https://github.com/crisr2/proyecto-monolito-naruto))
* Abrir la carpeta en donde esta el proyecto desde un IDE o desde CMD
* Construir el proyecto escribiendo el comando **./gradlew build** en la terminal
* Ejecutar el proyecto escribiendo el comando **./gradlew bootRun**
* Alternativamente se puede ejecutar el archivo **NarutoApplication.java** desde un IDE con el proyecto abierto para ejecutar el proyecto

La aplicación queda abierta en la dirección http://localhost:8080/
